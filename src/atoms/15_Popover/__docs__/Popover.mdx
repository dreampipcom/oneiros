# Popover

Popover component with different props.

#### Example

<Canvas of={Popover.Default} />

## Usage

```ts
import {Popover} from "@dreampipcom/oneiros";

const Example = () => {
  return (
      <Popover
        variant={EPopoverVariant.DEFAULT}
      >
        4.5/5
      </Popover>
  );
};

export default Example;
```

#### Arguments

- **children** - A string that represents the text contents of the Popover.
- **variant** - A Popover variant option (w/o Badge).
